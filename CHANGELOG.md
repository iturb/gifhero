# Installation
You can get GifHero easily with CocoaPods.

Add to your Podfile
```
pod 'GifHero'
```
Run installation
```
> pod install
```
If you are wondering what is CocoaPods take a look at: [https://cocoapods.org](https://cocoapods.org)

# Update

Run update
```
> pod update
```
